# 🚀 DB Guardian AI

## Real-Time Database Downtime Prediction & Monitoring System

DB Guardian AI is a real-time monitoring system that predicts potential database downtime using Machine Learning (Isolation Forest). It continuously monitors CPU, Memory, and Disk usage and triggers alerts when abnormal behavior is detected.

---

## 🌟 Features

- Real-time system monitoring (CPU, Memory, Disk)
- Machine Learning anomaly detection
- Live dashboard with charts
- Alert system for downtime risk
- Activity logs
- Downloadable performance reports (CSV)
- Cloud deployable (Render)

---

## 🧠 Technologies Used

- Python (Flask)
- Scikit-learn
- Pandas
- Psutil
- Chart.js
- Gunicorn
- HTML, CSS, JavaScript

## ⚙️ Local Setup

### Clone Repository

```bash
git clone https://github.com/DeepikaReddy0717/DB_GUARDIAN-AI-.git
cd DB_GUARDIAN-AI-
```

---

### Create Virtual Environment

```bash
python -m venv .venv
.venv\Scripts\activate
```

---

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

### Run Application

```bash
python app.py
```

Open in your browser:

```
http://127.0.0.1:5000
```
